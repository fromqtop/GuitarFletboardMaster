# 🎸 Guitar Fretboard Master

ギター指板を視覚的に学びながら、楽譜読解・音感トレーニングができるWebアプリです。

🌐 **Live Demo:**  
👉 https://fromqtop.github.io/GuitarFletboardMaster/

📰 **How to Play:**  
👉 https://web-breeze.net/guitar-fretboard-master-how-to-play/
