## 補足
このアプリは2020年7月にAndroidアプリとしてGoogle Playに公開していたものをWeb版にしたものです。  
（現在は開発およびAndroidアプリ版の公開は停止しており、更新の予定はありません）

当時はよくわからずとも完成することを最優先で開発していたため、Vueを使っているのにSPAにもせず、コンポーネント分割もせず、全ページに同じコードが多数登場・・・という感じですので、いつか作り直したいとは思っています・・・。
